# shit-ppy
Shit eater portfolio

### Запуск проекта 

1. Клонируем репозиторий

```bash
   git clone https://github.com/ShitCoderz/shit-ppy.git
```

2. Устанавливаем зависимости

```bash
   npm install
```

3. Запускаем режим разработки

```bash
   npm run dev
```
