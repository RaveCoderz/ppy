# shit-ppy
### Shit eater portfolio

## Запуск проекта

Клонируем репозиторий

```bash
   git clone https://github.com/ShitCoderz/shit-ppy.git
```

Устанавливаем зависимости

```bash
   npm install
```

Запускаем режим разработки

```bash
   npm run dev
```
